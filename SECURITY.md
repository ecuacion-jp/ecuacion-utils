# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ecuacion-utils, **please do not open a public GitHub Issue**, as this exposes the vulnerability before it can be fixed.

Instead, please report it privately via one of the following:

- **GitHub Private Security Advisory**: Use the [Report a vulnerability](https://github.com/ecuacion-jp/ecuacion-utils/security/advisories/new) button on the Security tab.
- **Email**: Send a report to [info@ecuacion.jp](mailto:info@ecuacion.jp).

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept (if applicable)
- The affected version(s)

We will respond as promptly as possible and work to address confirmed vulnerabilities in a timely manner.
