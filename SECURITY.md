# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ecuacion-utils, please report it by opening a [GitHub Issue](https://github.com/ecuacion-jp/ecuacion-utils/issues).

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept (if applicable)
- The affected version(s)

We will respond as promptly as possible and work to address confirmed vulnerabilities in a timely manner.
