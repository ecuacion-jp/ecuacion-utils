## Project Overview

A utility library for Excel and PDF manipulation using Apache POI and PDFBox. Multi-module Maven project with Jakarta EE support.

- **Java**: 21
- **Build tool**: Maven
- **Main modules**: `ecuacion-util-excel-table`, `ecuacion-util-excel-report-to-pdf`
